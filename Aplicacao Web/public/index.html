Esse sistema é baseado na Web das Coisas para Assistência Médica e Monitoramento Remoto de Pacientes.

O Ambientes de Vida Assistida tem uma forte relação com o conceito Internet das Coisas.

A Internet das Coisas possibilita que qualquer dispositivo do cotidiano esteja conectado à Internet, tornando-se um transmissor e receptor de informações em rede.

Um conceito importante da Internet das Coisas é a Web das Coisas, o qual propõe a adoção de padrões Web, a fim de oferecer uma linguagem comum entre os dispositivos e coisas do mundo físico.
